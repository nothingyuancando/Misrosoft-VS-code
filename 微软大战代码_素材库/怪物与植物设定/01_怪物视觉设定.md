# 01 怪物视觉设定

怪物采用「统一身体 + 梗图贴纸头 + 技能前摇」的设计。这样画面不会散，玩家远看能分辨类型，近看能看到梗。

## 视觉分层

```txt
身体剪影：决定怪物功能，例如小兵、快车、气泡、云、巨人。
头部贴纸：决定梗来源，例如更新进度、Edge、Teams、OneDrive、蓝屏。
技能前摇：决定玩家反应窗口，例如读条、会议圈、同步复制、stop code。
```

## 第一版怪物

### Windows 更新兵

资源建议：

```txt
头部贴纸：图片资源/怪物贴纸/windows_update_head.svg
系统 UI：图片资源/系统UI梗图/update_progress_37.svg
备用图标：图片资源/怪物贴纸/devicon_windows8_original.svg
```

外观：

蓝色小兵，抱着一个更新进度牌。头部贴纸显示 `37%` 或 Windows 方块。

动作：

```txt
行走：慢慢向左挪
前摇：进度条从 37% 涨到 100%
技能：本行代码植物卡顿 1 秒
```

战报：

```txt
Windows 更新兵表示，请不要关闭你的电脑。
```

### 99% 卡住怪

资源建议：

```txt
头部贴纸：图片资源/怪物贴纸/windows_update_head.svg
系统 UI：图片资源/系统UI梗图/update_progress_99.svg
```

外观：

身体是一个很厚的进度条窗口，头部永远显示 `99%`。

动作：

```txt
低血量时短暂无敌
无敌期间显示 Completing update...
被 rollback 命中后进度倒退
```

战报：

```txt
99% 卡住怪卡住了。它对此没有任何歉意。
```

### 强制重启车

资源建议：

```txt
头部贴纸：图片资源/怪物贴纸/restart_head.svg
系统 UI：图片资源/系统UI梗图/restart_prompt_mock.svg
```

外观：

小车，车头贴电源符号，车身写 `Restart now`。速度快，血量低。

动作：

```txt
撞到植物后让它进入重启状态
重启状态显示旋转加载圈
```

战报：

```txt
强制重启车没有找到“稍后”按钮。
```

### Edge 默认浏览器传教士

资源建议：

```txt
头部贴纸：图片资源/怪物贴纸/edge_head.svg
```

外观：

西装小人，头是蓝绿色浏览器旋涡贴纸，手里举着 `Set default?` 牌子。

动作：

```txt
靠近植物后开始劝说读条
读条满后植物进入默认浏览器状态
被 sudo 或 try-catch 打断
```

战报：

```txt
Edge 传教士正在询问 print 炮是否愿意成为默认输出方式。
```

### Teams 会议怪

资源建议：

```txt
头部贴纸：图片资源/怪物贴纸/teams_head.svg
```

外观：

紫色会议气泡，头上挂 `Meeting in 1 min` 通知。

动作：

```txt
展开紫色会议圈
圈内植物停止攻击
会议圈里飘出麦克风、摄像头和共享屏幕小图标
```

战报：

```txt
Teams 会议怪发起了一个 45 分钟会议。
for 循环枪表示自己还有一个问题。
```

### OneDrive 同步怪

资源建议：

```txt
头部贴纸：图片资源/怪物贴纸/onedrive_head.svg
```

外观：

云朵背包怪，背后拖着文件副本。

动作：

```txt
每隔几秒复制一个冲突副本
复制体名字显示在头顶
```

副本命名：

```txt
final.zip
final-copy.zip
final-copy-conflict.zip
```

战报：

```txt
OneDrive 同步怪制造了一个冲突副本。没人知道哪个才是最新版本。
```

### 蓝屏巨人

资源建议：

```txt
头部贴纸：图片资源/怪物贴纸/bsod_head.svg
系统 UI：图片资源/系统UI梗图/bsod_stopcode.svg
```

外观：

巨大蓝色窗口巨人，脸是 `:(`，胸口滚动 stop code。

动作：

```txt
每走 3 格读条一次
读条结束释放 stop code
本行植物短路
函数炮或 try-catch 可以打断
```

技能名：

```txt
MEMORY_MANAGEMENT
CRITICAL_PROCESS_DIED
IRQL_NOT_LESS_OR_EQUAL
PAGE_FAULT_IN_NONPAGED_AREA
```

战报：

```txt
蓝屏巨人释放了 MEMORY_MANAGEMENT。你的内存开始怀疑自己。
```

### Clippy 纸夹幽灵

资源建议：

```txt
头部贴纸：图片资源/怪物贴纸/clippy_head.svg
```

外观：

回形针幽灵，身体是半透明弹窗。

动作：

```txt
随机跳到植物旁边
弹出 Need help?
让植物本回合攻击错误目标
```

战报：

```txt
Clippy 纸夹幽灵问，看起来你正在防守，是否需要帮助？
```

## 后续怪物

后续可以继续加：

```txt
GWX 升级弹窗：会走路的升级弹窗
开始菜单广告商：占用种植格
Recall 快照眼：复制玩家最近操作
Windows 8 磁贴墙：移动盾牌怪
1809 文件吞噬者：偷走植物配置
```

这些怪都可以沿用同一套规则：身体负责功能，贴纸负责梗，技能前摇负责可玩性。
