# 02 代码植物视觉设定

代码植物不是传统植物，而是长在 VSCode 里的「代码盆栽」和「桌面小工具」。基础植物用咖啡种下，逻辑模块用内存叠加到已有植物上。

## 视觉分层

每个格子最多显示三层：

```txt
底层：基础植物，例如 print 炮、coffee 机、if 墙。
中层：逻辑模块 1，例如 for、if、try-catch。
上层：逻辑模块 2，例如 else、cache、sudo。
```

模块不要画成完整新植物。它们应该像插件、环、贴片、护盾、标签一样挂在基础植物上。

## 第一版基础植物

### coffee 机

资源建议：

```txt
图片资源/植物与代码图标/plant_coffee_machine.svg
下载备用：simpleicons_coffeescript.svg
```

外观：

小咖啡机，杯子里冒出代码蒸汽。产出咖啡时，杯子上方弹出 `+25`。

组合表现：

```txt
coffee + for：连续倒咖啡，但机器会冒烟
coffee + cache：第一次产出翻倍，杯子旁边出现缓存盒
coffee + try-catch：外面套花括号护盾
```

### print 炮

资源建议：

```txt
图片资源/植物与代码图标/plant_print_cannon.svg
```

外观：

一个命令行窗口长出炮口，炮口写 `print()`。发射 `{}`、`;`、`>` 字符弹。

组合表现：

```txt
print + for：炮口变成三连管，外面旋转 for 环
print + if：炮口前出现条件滤镜
print + try-catch：炮塔外套花括号护盾
print + for + try-catch：稳定三连炮
```

### if 墙

资源建议：

```txt
图片资源/植物与代码图标/plant_if_wall.svg
```

外观：

一扇条件门，门上写 `if {}`。敌人撞上时显示 `false`。

组合表现：

```txt
if + else：门后弹出备用反击板
if + try-catch：门框外出现异常保险罩
```

### bug 雷

资源建议：

```txt
图片资源/植物与代码图标/plant_bug_mine.svg
```

外观：

红色错误图标地雷，埋在格子里。触发时先弹出 `Error!`，再爆炸。

组合表现：

```txt
bug + for：连续爆炸
bug + rollback：爆炸后出现倒带箭头，把敌人推回去
bug + try-catch：减少提前爆炸概率
```

### rollback 器

资源建议：

```txt
图片资源/植物与代码图标/plant_rollback.svg
下载备用：devicon_git_original.svg
```

外观：

一个 git 回退箭头炮，像小型倒带机。

组合表现：

```txt
rollback + if：只瞄准精英怪
rollback + for：连续回滚多个敌人
```

### 函数炮

资源建议：

```txt
图片资源/植物与代码图标/plant_function_cannon.svg
```

外观：

坐标纸炮台，炮口出现当前函数曲线。

弹道按钮：

```txt
y = 0
y = x
y = -x
y = x*x
y = sin(x)
y = random()
```
