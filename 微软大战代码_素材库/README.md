# 微软大战代码素材库

这个目录用于存放游戏原型需要的视觉资源、怪物设定和植物设定。

## 目录结构

```txt
图片资源/
  怪物贴纸/          微软军团怪物头部贴纸、产品图标、梗图转绘
  植物与代码图标/    代码植物、逻辑模块、程序员工具图标
  战斗地点_VSCode/   VSCode 战场参考图和本地战场草图
  系统UI梗图/        蓝屏、更新、弹窗、广告等 UI 梗图
  来源记录/          下载清单和素材来源

怪物与植物设定/
  01_怪物视觉设定.md
  02_代码植物视觉设定.md
  03_VSCode战场设定.md
  04_贴图拼贴规范.md
  05_资源使用清单.md
```

## 当前资源策略

原型阶段采用两类资源：

- 下载图标，用于 VSCode、Windows、Git、Docker、JavaScript、TypeScript、npm、Stack Overflow 等识别度高的符号。
- 本地贴纸 SVG，用于 Windows 更新、蓝屏、Edge 传教士、Teams 会议怪、OneDrive 同步怪、Clippy 纸夹幽灵等游戏化角色。

这样做的好处是，原型能马上使用，不依赖一堆风格不统一的截图。后续如果找到更经典的梗图，可以替换同名贴纸文件。

## VSCode 战场方向

战斗地点可以放在 VSCode 上。推荐布局：

```txt
左侧资源管理器：植物卡牌栏
中间编辑器：5 行 x 9 列战场
底部终端：战报
右侧迷你地图：波次和怪物预警
顶部标签页：关卡名，例如 main.ts、defense.js、final_project.zip
```

这个设定和游戏主题很搭。玩家不是在普通草坪上防守，而是在自己的编辑器里防守项目文件。
