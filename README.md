# 微软大战代码：请不要关闭你的电脑

一个程序员梗向的横版塔防游戏策划与 HTML 原型。

玩家扮演深夜赶工的程序员，在类似 VS Code 的桌面战场上部署代码植物，抵抗从右侧入侵的微软军团。玩法参考《植物大战僵尸》的放置、防线推进和波次压力，并加入代码逻辑组合、微软系梗怪物、Graphwar 式函数炮弹道，以及程序员圈常见的 bug 和工具梗。

## 项目状态

当前仓库包含：

- 一套完整的中文游戏策划文档。
- 一个可直接打开的单页 HTML 原型。
- 一批 SVG 视觉素材、怪物贴纸、代码植物图标和 UI 梗图。

第一版目标是验证核心玩法：放置代码植物、使用咖啡和内存资源、叠加逻辑模块、抵御 5 波微软怪物，并击败最终 Boss 蓝屏巨人。

## 快速试玩

无需安装依赖，直接用浏览器打开：

```txt
微软大战代码_HTML原型/index.html
```

如果浏览器因为本地文件策略限制资源加载，也可以在项目根目录启动一个静态服务器：

```bash
python3 -m http.server 8000
```

然后访问：

```txt
http://localhost:8000/微软大战代码_HTML原型/
```

## 玩法概览

核心循环：

1. 使用 coffee 机生产咖啡。
2. 用咖啡种植基础代码植物。
3. 用内存给植物叠加逻辑模块。
4. 微软军团按波次从右侧入侵。
5. 代码植物自动攻击、阻挡、控制或生产资源。
6. 玩家根据敌人技能调整防线。
7. 最后一波击败蓝屏巨人，守住 main 分支。

代表性组合：

| 组合 | 效果 |
| --- | --- |
| `print + for` | 连续发射字符弹 |
| `print + if` | 优先攻击快怪或精英怪 |
| `print + try-catch` | 免疫一次打断 |
| `if 墙 + else` | 墙体被破坏后触发反击 |
| `bug 雷 + rollback` | 爆炸并击退敌人 |
| `rollback + if` | 优先回滚精英怪 |

## MVP 内容

第一版原型建议控制在一关内：

- 地图：5 行 x 9 列。
- 资源：咖啡、内存。
- 植物：coffee 机、print 炮、if 墙、bug 雷、rollback 器、函数炮。
- 模块：for、if、else、try-catch。
- 敌人：Windows 更新兵、强制重启车、Edge 默认浏览器传教士、Teams 会议怪、OneDrive 同步怪、蓝屏巨人。
- 波次：5 波，最终波出现蓝屏巨人。
