# 01 核心玩法

## 游戏类型

《微软大战代码：VS Code 桌面幸存者》是一个程序员梗向的移动生存游戏。

玩家控制 VS Code 图标，在编辑器窗口中移动、躲避、捡 coffee、升级植物武器，并用锦囊牌处理微软弹窗和开发事故。游戏不再使用横版路线和格子种植，而是让敌人从编辑器边缘涌入，让弹窗直接污染战场。

## 战场结构

第一版战场是一个 VS Code 风格界面：

```txt
┌─────────────────────────────────────────────┐
│ titlebar: 微软大战代码：VS Code 桌面幸存者   │
├───────────────┬─────────────────┬───────────┤
│ Explorer      │ tabs/status      │ Battle Kit│
│ file hp       ├─────────────────┤ skills    │
│ already       │ tricks hand      │ weapons   │
│ deleted state ├─────────────────┤ events    │
│               │ editor arena     │ objective │
│               │ VS Code player   │           │
│               │ enemies/popups   │           │
├───────────────┴─────────────────┴───────────┤
│ Terminal survivor log                         │
└─────────────────────────────────────────────┘
```

中间编辑器区域是主战场。玩家控制 VS Code 图标在其中移动。敌人从四周刷出，弹窗会覆盖在战场上。

左侧 Explorer 显示项目文件生命。文件生命归零后显示 `already deleted`，但真实磁盘文件不会被删除。

## 胜负条件

胜利条件：

```txt
撑到 3 分钟
击败蓝屏合并冲突 Boss
至少仍有项目文件处于 safe
```

失败条件：

```txt
VS Code 主角生命归零
或者所有文件都进入 already deleted
```

失败弹窗文案：

```txt
你的代码文件被删掉了
这是游戏里的模拟事故。真实磁盘文件没有被删除，也不会被修改。
```

## 局内循环

基础循环：

```txt
1. 玩家移动 VS Code 图标躲避事故
2. 植物武器自动攻击周围敌人
3. 击败敌人掉落 coffee
4. 拾取 coffee 获得资源和经验
5. 经验满后弹出 3 个植物武器升级选项
6. 玩家用上方锦囊手牌救场
7. 微软弹窗周期性出现，未关闭会伤害文件
8. 3 分钟后出现蓝屏合并冲突 Boss
```

## 操作方式

操作要保持轻量、直接、爽快：

```txt
WASD / 方向键    移动 VS Code 主角
Space            Ctrl+S 保存冲击波
Q                kill -9 最危险事故
E                Alt+F4 关闭弹窗
鼠标点击手牌      打出锦囊牌
鼠标点击弹窗      手动关闭弹窗
```

玩家不需要输入真实命令，不需要修真实代码。程序员元素通过武器、卡牌、敌人和文本表达。

## 主角和武器

主角是 VS Code 图标。已解锁的植物武器会显示在主角周围，并带有等级角标。

开局默认武器：

```txt
print 炮 Lv.1
```

升级池包含：

```txt
coffee 机
print 炮
bug 雷
rollback
函数炮
Docker
TypeScript
JavaScript
Python
IDEA
Tomcat
Spring Boot
```

## 锦囊手牌

手牌保留三国杀式的救场节奏。抽卡和打出卡牌都消耗 coffee。

第一版手牌：

```txt
Ctrl+S：保存冲击波
kill -9：秒杀或重创高威胁敌人
Stack Overflow：全屏答案弹幕，但招来隐藏 bug
Git Stash：短暂无敌并恢复危险文件
Docker Build：立刻获得容器护盾
热咖啡：回血并获得 coffee
CI Pipeline：延迟全屏扫描小怪
Alt+F4：关闭全部弹窗
```

## 敌人与弹窗

敌人不是“僵尸”，而是被夸张化的桌面事故：

```txt
Windows 更新
Edge 弹窗
Teams 会议
OneDrive 冲突
npm 黑洞
Defender 误杀
YAML 缩进
需求变更
蓝屏合并冲突
```

弹窗是特殊干扰。它们会遮挡战场并倒计时。玩家可以点掉，也可以用 `Alt+F4` 清掉。如果倒计时结束还没处理，弹窗会污染一个文件。

## 战报系统

底部 Terminal 显示战斗日志。战报不做复杂教程，只解释关键事件并保留梗味。

示例：

```txt
VS Code 图标已经进入编辑器战场。
植物现在是武器：print 炮会自动开火。
桌面幸存者模式启动。别让微软事故把工作区盖满。
OneDrive 同步怪复制了一个冲突副本。
Alt+F4 关闭 3 个弹窗，Edge 的话术被打断。
蓝屏合并冲突巨人出现：屏幕开始怀疑人生。
```

## 特色机制

当前版本最核心的特色是：

```txt
VS Code 主角
+ 植物武器构筑
+ 锦囊抽卡
+ 弹窗大乱斗
+ 文件假删除
```

它保留程序员梗，但避免做成真实开发模拟；保留植物，但不再落回塔防种植框架。
