# 02 资源与组合系统

## 资源设计

当前版本只保留一种主资源：

```txt
coffee
```

coffee 同时承担三件事：

```txt
抽锦囊牌
打出锦囊牌
推动经验和升级节奏
```

旧版本的“内存”资源已经移除。原因是当前玩法从塔防改为移动生存，资源太多会让玩家在躲避、拾取、抽卡、升级之间分心过重。第一版先让 coffee 作为统一节奏资源。

## coffee 来源

coffee 来源：

```txt
击败敌人后掉落 coffee
拾取 coffee 后获得资源与经验
coffee 机武器周期性额外产出
热咖啡锦囊立刻补充
Boss 和精英敌人掉落更多 coffee
```

coffee 掉落既是资源，也是引导玩家移动的“诱饵”。玩家为了升级必须靠近战场风险区拾取掉落物。

## coffee 用途

coffee 用途：

```txt
抽一张锦囊牌
打出手牌
部分卡牌产生额外收益或回血
```

当前数值：

```txt
抽卡费用：25 coffee
手牌上限：4
开局免费摸 2 张
```

## 经验与升级

拾取 coffee 时会增加经验。经验满后暂停战斗，弹出 3 个升级选项。

升级选项来自植物武器池：

```txt
未拥有：解锁该武器
已拥有：提升该武器等级
```

玩家选择后继续战斗。

## 武器构筑

植物现在是武器，而不是种植单位。

当前武器池：

| 武器 | 资源定位 | 作用 |
| --- | --- | --- |
| `print 炮` | 默认输出 | 稳定字符弹 |
| `coffee 机` | 经济/恢复 | 提高收益并回血 |
| `bug 雷` | 陷阱爆发 | 周期性布置地雷 |
| `rollback` | 控场 | 推开附近事故 |
| `函数炮` | 特殊弹道 | 曲线弹 |
| `Docker` | 防御 | 容器护盾 |
| `TypeScript` | 精准输出 | 锁定高威胁目标 |
| `JavaScript` | 高频输出 | 随机飞镖 |
| `Python` | 范围清理 | 自动化扫场 |
| `IDEA` | 单点爆发 | 重构激光 |
| `Tomcat` | 正面压制 | 8080 火球 |
| `Spring Boot` | 恢复 | 健康检查回血和修文件 |

## 主角挂载显示

已解锁武器会显示在 VS Code 主角周围：

```txt
图标：武器素材
角标：当前等级
位置：单武器显示在主角右侧，多武器围绕主角排列
```

这个视觉反馈很重要。玩家需要立刻知道自己当前“装了什么”。

## 锦囊系统

锦囊牌负责救场和爆发，不能替代武器长期输出。

当前手牌池：

| 锦囊 | 费用 | 定位 |
| --- | ---: | --- |
| `Ctrl+S` | 18 | 保存冲击波，击退附近敌人 |
| `kill -9` | 28 | 终止最危险事故 |
| `Stack Overflow` | 22 | 全屏弹幕，但召唤隐藏 bug |
| `Git Stash` | 20 | 短暂无敌，恢复危险文件 |
| `Docker Build` | 24 | 立刻获得护盾，升级 Docker |
| `热咖啡` | 10 | 回血并补 coffee |
| `CI Pipeline` | 30 | 延迟全屏扫描小怪 |
| `Alt+F4` | 18 | 关闭全部弹窗 |

## 快捷锦囊

为了保证玩家即使没抽到关键牌也有基本救场能力，右侧提供 3 个冷却技能：

```txt
Space：Ctrl+S
Q：kill -9
E：Alt+F4
```

这些技能不消耗手牌，但有冷却。手牌版同名效果则消耗 coffee，可以在冷却之外额外使用。

## 组合方向

当前 HTML 原型还没有正式“进化合成”，但后续可以加入以下组合：

| 组合 | 进化名 | 效果 |
| --- | --- | --- |
| `Docker + CI Pipeline` | 自动化部署轨道炮 | 周期性全屏扫描 |
| `TypeScript + JavaScript` | 严格模式热修 | 高频弹幕附带减速 |
| `Git Stash + rollback` | 时间线保险 | 受致命伤时回到几秒前 |
| `Python + Stack Overflow` | 复制粘贴脚本军团 | 召唤脚本分身，偶尔出 bug |
| `Spring Boot + Tomcat` | 后端服务堡垒 | 回复区域和 8080 火墙 |
| `函数炮 + bug 雷` | 混沌曲线炸弹 | 曲线弹命中后爆炸 |

## 平衡原则

第一版平衡目标：

```txt
武器负责长期生存
锦囊负责瞬时转折
coffee 既要能抽卡，也要能诱导玩家冒险拾取
弹窗必须烦，但不能让玩家完全失控
文件 already deleted 要紧张，但不能太早雪崩
```
