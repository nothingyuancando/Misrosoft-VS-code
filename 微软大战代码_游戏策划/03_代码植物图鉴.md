# 03 代码植物图鉴

## 设计原则

代码植物要做到三件事：

- 名字让程序员会心一笑。
- 功能让非程序员也能看懂。
- 组合后产生明显变化。

基础植物用咖啡种植。逻辑模块用内存叠加。部分单位既可以作为基础植物，也可以作为模块，但第一版建议保持简单。

## 基础植物

### coffee 机

```txt
类型：资源植物
费用：50 咖啡
效果：每 8 秒生产 25 咖啡
组合槽：1
```

说明：

coffee 机是向日葵位。它不攻击，但决定玩家经济。

可组合：

```txt
coffee + cache：首次产出翻倍，后续有概率产出“冷咖啡”
coffee + for：连续产出 3 次，然后进入疲劳
coffee + try-catch：免疫一次 Teams 会议或 Edge 转化
```

战报：

```txt
coffee 机产出了一杯救命咖啡。
```

### print 炮

```txt
类型：基础输出
费用：100 咖啡
效果：每 1.5 秒向前发射 1 颗字符弹
组合槽：2
```

说明：

print 炮是豌豆射手位。它简单、稳定，是大部分组合的基础。

可组合：

```txt
print + for：连续发射
print + if：优先攻击特定目标
print + try-catch：免疫一次打断
print + async：延迟发射强力弹
```

战报：

```txt
print 炮输出了一颗字符弹。
```

### if 墙

```txt
类型：防御植物
费用：75 咖啡
效果：阻挡敌人前进
组合槽：2
```

说明：

if 墙负责挡路。它可以和 else 组合成反击墙，也可以和 try-catch 组合成抗异常防线。

可组合：

```txt
if + else：被摧毁后触发反击
if + try-catch：抵消一次特殊技能
if + regex：只允许某类敌人通过，其他敌人减速
```

战报：

```txt
if 墙判断敌人不该通过。
```

### bug 雷

```txt
类型：一次性陷阱
费用：125 咖啡
效果：敌人踩到后爆炸
组合槽：1
```

说明：

bug 雷是高伤害陷阱。适合处理快怪、精英怪和漏网怪。

可组合：

```txt
bug + for：连续爆炸 3 次
bug + rollback：爆炸后击退敌人
bug + try-catch：防止提前引爆
```

战报：

```txt
bug 雷被触发了。没人知道为什么，但它炸得很响。
```

### rollback 器

```txt
类型：控制植物
费用：150 咖啡
效果：每 10 秒把最近敌人推回 3 格
组合槽：2
```

说明：

rollback 器是关键控场单位。它能拖住高血量怪，也能处理 99% 卡住怪。

可组合：

```txt
rollback + if：只回滚精英怪
rollback + for：连续回滚多个敌人
rollback + bug：造成时空爆炸
```

战报：

```txt
git rollback 成功，敌人回到了刚才的位置。
```

### regex 藤蔓

```txt
类型：控制植物
费用：125 咖啡
效果：定身一名敌人 2 秒
组合槽：2
```

说明：

regex 藤蔓强力但不稳定。它是程序员梗味最浓的控制单位。

可组合：

```txt
regex + for：多次定身
regex + if：只匹配指定敌人
regex + try-catch：匹配失败时不误伤
```

战报：

```txt
regex 藤蔓开始匹配。所有人都看不懂它写了什么。
```

### scan 雷达

```txt
类型：侦测植物
费用：100 咖啡
效果：攻击隐身和飞行敌人，提前发现 boss 技能
组合槽：1
```

说明：

scan 雷达用于处理蓝牙驱动幽灵、Copilot 小飞机、蓝屏巨人的 stop code。

可组合：

```txt
scan + if：优先标记高威胁敌人
scan + cache：提升附近植物命中率
```

战报：

```txt
scan 雷达发现了一个不该存在的进程。
```

### 函数炮

```txt
类型：特殊攻击植物
费用：200 咖啡
发射消耗：60 内存
效果：按函数轨迹发射炮弹
组合槽：0
```

说明：

函数炮是 Graphwar 元素的承载单位。它不要求玩家写复杂公式，第一版用按钮选择弹道。

可选弹道：

```txt
y = 0：直线
y = x：斜线
y = -x：反向斜线
y = x*x：抛物线
y = sin(x)：波浪
y = random()：乱码
```

战报：

```txt
函数炮载入 y = sin(x)，炮弹开始蛇形走位。
```

## 逻辑模块

### for 模块

```txt
类型：输出增强
费用：40 内存
效果：让目标植物连续触发 3 次
副作用：多次触发后过热
```

典型组合：

```txt
print + for = for-print 循环炮
bug + for = 连环 bug 雷
rollback + for = 多段回滚器
```

### if 模块

```txt
类型：条件判断
费用：30 内存
效果：让目标植物优先处理指定敌人
副作用：没有满足条件的敌人时，效率降低
```

可选条件：

```txt
优先攻击最快敌人
优先攻击血量最高敌人
优先攻击正在施法的敌人
优先攻击本行最靠左敌人
```

### else 模块

```txt
类型：反击逻辑
费用：35 内存
效果：当前置条件失败或植物被摧毁时，触发一次备用效果
```

典型组合：

```txt
if 墙 + else = if-else 双分支墙
print + if + else = 条件没命中时改打最近敌人
```

### try-catch 模块

```txt
类型：保护逻辑
费用：50 内存
效果：抵消一次异常、会议、蓝屏、弹窗或转化
冷却：20 秒
```

它是中后期必备模块。

### cache 模块

```txt
类型：加速逻辑
费用：35 内存
效果：提升首次攻击或首次产出
副作用：可能使用旧数据
```

### async/await 模块

```txt
类型：高级控制
费用：70 内存
效果：延迟后释放更强效果
副作用：被打断时触发未处理异常
```

### sudo 模块

```txt
类型：权限强化
费用：60 内存
效果：提高伤害或驱散权限类敌人技能
副作用：被权限弹窗怪克制时会停顿
```

## 推荐第一版植物池

第一版只做这些：

```txt
基础植物：coffee 机、print 炮、if 墙、bug 雷、rollback 器、函数炮
逻辑模块：for、if、else、try-catch
```

这样已经可以形成核心组合：

```txt
print + for
print + if
print + for + try-catch
if + else
bug + rollback
rollback + if
```
