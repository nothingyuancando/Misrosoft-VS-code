# 03 代码植物图鉴

## 设计原则

代码植物已经从“格子里的防御塔”改为“VS Code 主角的自动武器”。

每个植物武器需要做到：

```txt
一眼能看出程序员梗
攻击表现足够清楚
升级后有明显变强
图标能挂在 VS Code 主角周围
```

## 第一版武器

### print 炮

```txt
定位：默认主武器
表现：自动向最近敌人发射字符弹
开局：默认拥有 Lv.1
```

升级方向：

```txt
提高射速
增加散射数量
提升字符弹伤害
```

战报：

```txt
植物现在是武器：print 炮会自动开火。
```

### coffee 机

```txt
定位：经济与恢复
表现：周期性提供 coffee，并小幅恢复主角生命
```

升级方向：

```txt
提高 coffee 收益
缩短产出间隔
增加回血量
```

设计意图：

coffee 机不是输出武器，而是构筑里的经济引擎。玩家选它等于牺牲短期火力换长期成长。

### bug 雷

```txt
定位：陷阱爆发
表现：周期性在主角附近丢出 bug 雷，敌人靠近后爆炸
```

升级方向：

```txt
提高爆炸伤害
扩大爆炸半径
缩短布雷间隔
```

战报：

```txt
bug 雷被触发了。没人知道为什么，但它炸得很响。
```

### rollback

```txt
定位：控场
表现：定期释放回滚脉冲，把附近敌人推远并造成伤害
```

升级方向：

```txt
扩大脉冲范围
提高击退距离
缩短触发间隔
```

设计意图：

rollback 是被包围时的空间控制武器。它让玩家有机会从弹窗和敌人堆里脱身。

### 函数炮

```txt
定位：特殊弹道
表现：发射带曲线偏移的函数弹
```

升级方向：

```txt
提高曲线弹伤害
缩短发射间隔
增加曲线弹尺寸或穿透
```

后续可扩展：

```txt
允许选择 sin / cos / tan / random 等弹道风格
```
