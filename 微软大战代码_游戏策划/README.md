# 微软大战代码：请不要关闭你的电脑

这是一个程序员梗向的横版塔防游戏策划。核心参考《植物大战僵尸》的直观放置、防线推进和波次压力，再加入代码逻辑组合、微软系梗怪物、Graphwar 式函数炮弹道，以及程序员圈常见的 bug 和工具梗。

玩家扮演深夜赶工的程序员，在电脑桌面上部署代码植物，抵抗从右侧入侵的微软军团。玩家用咖啡种植基础代码，用内存叠加逻辑模块，把简单代码组合成更强、更怪、更容易失控的复合防御。

## 核心定位

一句话玩法：

> 用 coffee 机、print 炮、if 墙、for 循环、try-catch 护盾和 git rollback，抵抗 Windows 更新兵、Edge 传教士、Teams 会议怪、OneDrive 同步怪和蓝屏巨人，守住 main 分支。

目标体验：

- 像《植物大战僵尸》一样一眼看懂，放塔、挡怪、守线。
- 像程序员梗图一样有记忆点，每个单位名字都自带笑点。
- 像写代码一样能组合，`print` 上叠 `for` 变连发，叠 `if` 变条件射击，叠 `try-catch` 变稳定输出。
- 像 Graphwar 一样偶尔有函数弹道，简单公式能打出离谱轨迹。

## 第一版目标

第一版不要做联网，不做复杂剧情，也不做真正代码编辑器。先验证三件事：

- 放置代码植物能形成清楚的塔防策略。
- 代码组合能带来爽感和意外，例如 `print + for`、`if + else`、`bug + rollback`。
- 微软怪物和战报足够欠打，让玩家愿意截图和复盘。

第一版建议做成网页 demo。地图 5 行 x 9 列，资源只有咖啡和内存，植物 8 个，怪物 6 个，关卡 1 个，目标是撑过 5 波并击败蓝屏巨人。
