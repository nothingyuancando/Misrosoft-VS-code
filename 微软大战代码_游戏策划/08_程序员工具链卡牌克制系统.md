# 08 程序员工具链卡牌克制系统

## 设计目标

这一阶段的目标是把“程序员工具链”做成幸存者构筑，而不是把真实开发流程搬进游戏。

核心体验：

```txt
微软大战代码 = VS Code 主角
+ 程序员工具武器
+ 微软桌面事故
+ 开发问题怪
+ 三国杀式锦囊救场
```

玩家不是真的修依赖、改配置、查日志，而是用这些程序员梗做成的武器和卡牌，在桌面事故里爽快生存。

## 三层玩法

### 1. 桌面幸存者层

```txt
控制 VS Code 图标移动
敌人从编辑器边缘涌入
植物武器自动攻击
coffee 掉落驱动升级
3 分钟后出现蓝屏合并冲突 Boss
```

### 2. 锦囊卡牌层

上方手牌表现“程序员临场救火”：

| 三国杀味道 | 程序员牌 | 当前效果 |
| --- | --- | --- |
| 闪 | `Ctrl+S` | 保存冲击波，击退附近事故 |
| 杀 | `kill -9` | 终止最危险事故 |
| 桃 | `热咖啡` | 回血并补 coffee |
| 锦囊 | `Stack Overflow` | 全屏答案弹幕，但招来隐藏 bug |
| 锦囊 | `Git Stash` | 短暂无敌并恢复危险文件 |
| 装备 | `Docker Build` | 立刻获得容器护盾并升级 Docker |
| 延时锦囊 | `CI Pipeline` | 3 秒后全屏扫描小怪 |
| 闪 | `Alt+F4` | 关闭全部弹窗 |

第一版规则：

```txt
开局摸 2 张
手牌上限 4 张
抽卡消耗 coffee
出牌消耗 coffee
点击手牌立刻生效
```

### 3. 工具链克制层

每个工具武器针对一类开发事故：

| 工具武器 | 解决的问题 | 克制对象 |
| --- | --- | --- |
| Docker | 环境隔离、容器护盾 | OneDrive 冲突、Defender 误杀 |
| TypeScript | 类型收口、配置约束 | npm 黑洞、YAML 缩进、需求变更 |
| JavaScript | 快速热修、前端机动 | Edge 弹窗、Teams 会议、npm 黑洞 |
| Python | 自动化脚本、批处理 | YAML 缩进、npm 黑洞、Defender 误杀 |
| IDEA | 重构、定位根因 | YAML 缩进、需求变更、npm 黑洞 |
| Tomcat | HTTP 容器、端口承压 | Windows 更新、蓝屏 Boss |
| Spring Boot | 健康检查、服务稳定 | 需求变更、YAML 缩进、蓝屏 Boss |
| 函数炮 | 曲线弹道、特殊爆发 | Boss、Edge 弹窗 |

## 问题怪视觉表现

问题怪应该像“开发事故实体化”，不是普通怪物换贴图。

| 问题怪 | 视觉关键词 | 行为 | 被克制时反馈 |
| --- | --- | --- | --- |
| npm 黑洞 | npm 标识、依赖链、版本碎片 | 慢速高血，压迫战场 | TS/Python/IDEA 命中像“依赖收敛” |
| YAML 缩进 | 配置纸片、错位缩进线 | 快速靠近，数量多 | TS/Python/IDEA 命中像“缩进修复” |
| Defender 误杀 | 安全盾、隔离十字 | 高伤害精英 | Docker 护盾像“容器隔离成功” |
| 需求变更 | 需求单、红色修改笔 | 慢速厚血 | IDEA/Spring Boot 命中像“需求冻结” |
| OneDrive 冲突 | 云同步、最终版副本 | 受伤后复制 | Docker/Python 清掉副本 |

## 弹窗卡牌互动

弹窗是特殊敌人：

```txt
Windows Update
Edge
Teams
OneDrive
BSOD
```

互动方式：

```txt
鼠标点击：关闭单个弹窗
Alt+F4：关闭全部弹窗
Ctrl+S：用冲击波争取走位空间
Docker Build：给自己护盾，顶过弹窗和敌人夹击
Git Stash：短暂无敌并救文件
```

## 武器进化方向

后续可以加入“双武器进化”，让构筑更像肉鸽。

| 条件 | 进化 | 效果 |
| --- | --- | --- |
| Docker + CI Pipeline | 自动化部署轨道炮 | 周期性扫描全屏 |
| TypeScript + JavaScript | 严格模式热修 | 高频弹幕附带减速 |
| Python + Stack Overflow | 脚本答案军团 | 召唤脚本弹幕，偶尔产生 bug |
| Spring Boot + Tomcat | 后端服务堡垒 | 回复光环和 8080 火墙 |
| 函数炮 + bug 雷 | 混沌曲线炸弹 | 函数弹命中后爆炸 |
| rollback + Git Stash | 时间线保险 | 低血时自动回滚一次 |

## 第一版实现范围

当前 HTML 原型已经实现：

```txt
VS Code 主角
12 个植物/工具武器
8 张锦囊手牌
9 类敌人或事故
5 类可点击弹窗
文件 already deleted 状态
武器挂载到主角周围
```

## 后续扩展

下一阶段建议：

- 给每个武器增加更明显的专属特效。
- 加入进化合成和稀有升级。
- 增加弃牌堆或临时手牌事件。
- 让 Boss 根据玩家最高等级武器改变技能。
- 给文件增加被动能力，例如 `package.json` 影响 npm 怪、`README.md` 影响升级质量。
