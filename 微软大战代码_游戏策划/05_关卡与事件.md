# 05 关卡与事件

## 关卡设计目标

关卡要让玩家逐步理解三个系统：

```txt
咖啡用于种植
内存用于组合
不同微软怪物需要不同代码逻辑处理
```

每关只引入一个主要新点。不要同时给玩家太多新植物和新怪物。

## 第一章：请不要关闭你的电脑

```txt
学会种 coffee 机和 print 炮
```

### 第 6 关，蓝屏降临

目标：

```txt
撑过 5 波
打败蓝屏巨人
守住 main 分支
```

解锁：

```txt

```

```txt

```

## 全局事件

全局事件在关卡中偶尔触发。它们会改变短时间局势。

### Windows 正在准备更新

```txt
效果：屏幕顶部出现更新进度条
进度到 100% 时，全场植物停顿 2 秒
玩家可以击败更新兵降低进度
```

战报：

```txt
Windows 正在准备更新，进度 37%。
```

### 老板来了

```txt
效果：全场暂停 3 秒，敌人和植物都不动
用途：给玩家喘息时间
触发：低概率随机事件，或作为玩家大招
```

战报：

```txt
老板来了。
所有进程开始假装正经。
```

### 网络波动

```txt
效果：OneDrive 同步怪停止复制，但玩家的函数炮命中率下降
```

战报：

```txt
网络波动。
云端和本地同时开始装傻。
```

### 蓝屏预警

```txt
效果：蓝屏巨人出现前 15 秒触发
提示玩家准备 try-catch 和 rollback
```

战报：

```txt
系统检测到一个令人熟悉的蓝色风险。
```

## 隐藏关卡

### 全球蓝屏风暴

灵感来自 2024 年全球 Windows 设备蓝屏事件。严格说这不是微软自己的 bug，可以作为隐藏事件或番外关。

目标：

```txt
在多行随机短路的情况下撑过 4 分钟
```

特殊敌人：

```txt
补丁乌云
安全模式幽灵
蓝屏裂缝
```

特殊植物：

```txt
安全模式启动盘
```

胜利提示：

```txt
你重启了 15 次。
奇迹发生了。
```

## 关卡节奏模板

每关可按这个节奏制作：

```txt
0:00 到 0:30，安全建设期，少量普通怪
0:30 到 1:30，引入本关主题怪
1:30 到 2:30，主题怪和基础怪混合
2:30 到 3:30，加入快怪或分裂怪制造压力
3:30 之后，精英怪或 boss 出现
```

第一版只做一个完整关卡时，可以把上述节奏压缩成 5 波。
