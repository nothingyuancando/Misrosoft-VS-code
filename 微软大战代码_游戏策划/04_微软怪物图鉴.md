# 04 微软怪物图鉴

## 设计原则

怪物不再沿固定路线前进，而是从编辑器边缘涌入，追逐 VS Code 主角。

怪物设计重点：

```txt
像桌面事故，不像普通僵尸
行为简单但有记忆点
能和武器形成克制关系
死亡后掉 coffee，推动升级节奏
```

## 当前敌人

### Windows 更新

```txt
定位：基础敌人
表现：带着更新进度靠近玩家
特点：稳定、常见、数量多
克制：print 炮、Tomcat
```

战报：

```txt
Windows 更新表示，请不要关闭你的电脑。
```

### Edge 弹窗

```txt
定位：弹窗召唤敌人
表现：靠近玩家，并周期性生成 Edge 弹窗
特点：干扰视野和文件安全
克制：JavaScript、Alt+F4、函数炮
```

战报：

```txt
Edge 的话术被打断。
```

### Teams 会议

```txt
定位：中速干扰敌人
表现：紫色会议怪追逐玩家
特点：逼玩家走位，配合弹窗制造压力
克制：JavaScript、Ctrl+S
```

战报：

```txt
Teams 会议怪发起会议：你能听到我吗？
```

### OneDrive 冲突

```txt
定位：分裂敌人
表现：受伤到一定程度后复制一个冲突副本
特点：如果输出太分散，会越打越乱
克制：Docker、Python、范围伤害
```

战报：

```txt
OneDrive 同步怪复制了一个冲突副本。
```

### npm 黑洞

```txt
定位：慢速高血敌人
表现：像 node_modules 黑洞一样压迫战场
特点：血量较高，适合测试单点和范围输出
克制：TypeScript、Python、IDEA
```

战报：

```txt
npm 黑洞开始解析依赖树。
```

### Defender 误杀

```txt
定位：精英干扰敌人
表现：带隔离和误杀主题的贴图
特点：伤害较高，适合逼出 Docker 护盾
克制：Docker、Python
```

战报：

```txt
Defender 误杀怪把缓存标记成可疑文件。
```

### YAML 缩进

```txt
定位：快速小怪
表现：缩进错乱的配置怪
特点：速度较快，数量上来后考验走位
克制：TypeScript、Python、IDEA
```

战报：

```txt
YAML 缩进怪把两个空格改成了四个空格。
```

### 需求变更

```txt
定位：精英厚血敌人
表现：产品需求变更实体化
特点：血量高，节奏慢，但压迫感强
克制：TypeScript、IDEA、Spring Boot
```

战报：

```txt
需求变更怪发来一句：这个很简单，顺手改一下。
```

### 蓝屏合并冲突

```txt
定位：最终 Boss
出现：约 3 分钟
表现：蓝屏巨人
特点：高血量，周期性生成 BSOD 弹窗，并伤害文件
克制：函数炮、Tomcat、Spring Boot、kill -9
```

战报：

```txt
蓝屏合并冲突巨人出现：屏幕开始怀疑人生。
```

## 弹窗怪

弹窗不是普通敌人，而是覆盖在战场上的桌面事件。

当前弹窗：

```txt
Windows Update：重启倒计时
Edge：默认浏览器推荐
Teams：会议邀请
OneDrive：同步最终版 zip
BSOD：蓝屏 stop code
```

规则：

```txt
可以鼠标点击关闭
可以用 Alt+F4 一键关闭
倒计时结束会伤害一个文件
```
