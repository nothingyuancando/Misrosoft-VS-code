# 04 微软怪物图鉴

## 设计原则

微软怪物来自真实用户吐槽、历史 bug、产品梗和办公软件记忆点。它们不需要严肃还原，重点是把玩家熟悉的烦人体验变成可见的塔防威胁。

每个怪物要有：

```txt
一眼能懂的外观
一个有梗的技能
一个明确弱点
一条欠打的战报
```

## 第一版怪物

### Windows 更新兵

```txt
定位：基础怪
速度：慢
血量：中
技能：走到半场后读条，读条结束让本行植物停顿 1 秒
弱点：print 炮、for-print 循环炮
```

外观：

抱着“正在更新 37%”牌子的蓝色小兵。

战报：

```txt
Windows 更新兵表示，请不要关闭你的电脑。
```

### 强制重启车

```txt
定位：快攻怪
速度：快
血量：低
技能：撞到第一个植物后，让它重启 3 秒
弱点：if 墙、debounce、rollback
```

外观：

刹车坏掉的小车，车头写着“稍后提醒我”。

战报：

```txt
强制重启车没有找到“稍后”按钮。
```

### Edge 默认浏览器传教士

```txt
定位：转化怪
速度：中快
血量：中
技能：靠近植物后开始劝说，读条结束让植物停止攻击并给敌人加速
弱点：sudo、try-catch、rollback
```

外观：

西装浏览器图标人，手里拿着“设为默认”牌子。

战报：

```txt
Edge 传教士正在询问 print 炮是否愿意成为默认输出方式。
```

### Teams 会议怪

```txt
定位：范围控制怪
速度：慢
血量：中
技能：展开会议圈，圈内植物停止攻击 3 秒
弱点：try-catch、coffee 加班、函数炮
```

外观：

紫色会议气泡，头顶写着“你有 13 人正在等待”。

战报：

```txt
Teams 会议怪发起了一个 45 分钟会议。
for 循环枪表示自己还有一个问题。
```

### OneDrive 同步怪

```txt
定位：分裂怪
速度：中
血量：低
技能：每隔几秒复制一个冲突副本
弱点：git backup、for-print 循环炮、regex
```

外观：

背着云朵包的小怪，身后拖着一串同名文件。

复制体命名：

```txt
项目最终版
项目最终版-副本
项目最终版-副本-冲突
```

战报：

```txt
OneDrive 同步怪制造了一个冲突副本。
没人知道哪个才是最新版本。
```

### 蓝屏巨人

```txt
定位：第一版 boss
速度：慢
血量：极高
技能：每走 3 格释放一次 stop code，让本行植物短路
弱点：scan 雷达、try-catch、函数炮、rollback
```

外观：

巨大的蓝色方块脸，脸上写着 `:(`。

技能名：

```txt
CRITICAL_PROCESS_DIED
MEMORY_MANAGEMENT
IRQL_NOT_LESS_OR_EQUAL
PAGE_FAULT_IN_NONPAGED_AREA
```

战报：

```txt
蓝屏巨人释放了 MEMORY_MANAGEMENT。
你的内存开始怀疑自己。
```

## 后续怪物

### 99% 卡住怪

```txt
定位：高血量拖延怪
速度：极慢
血量：高
技能：血量低于 20% 时进入短暂无敌，显示“正在完成更新”
弱点：rollback、函数炮
```

战报：

```txt
99% 卡住怪卡住了。
它对此没有任何歉意。
```

### GWX 升级弹窗

```txt
定位：弹窗干扰怪
速度：漂浮
血量：低
技能：第一次受到攻击时弹出确认框，附近植物有概率被预约升级
弱点：try-catch、esc 键、sudo
```

战报：

```txt
GWX 弹窗的关闭按钮看起来很安全。
这通常不是好事。
```

### 1809 文件吞噬者

```txt
定位：降级怪
速度：中
血量：中高
技能：经过植物旁边时偷走配置文件，让植物降级
弱点：git backup、try-catch
```

战报：

```txt
1809 文件吞噬者拿走了 Documents。
它说这是一次体验升级。
```

### Clippy 纸夹幽灵

```txt
定位：帮倒忙怪
速度：中
血量：低
技能：随机跳到植物旁边，让它本回合攻击错误目标
弱点：esc 键、try-catch
```

战报：

```txt
Clippy 纸夹幽灵问，看起来你正在防守，是否需要帮助？
```

### Recall 快照眼

```txt
定位：复制玩家套路的怪
速度：慢
血量：中
技能：记录最近放置的植物，生成敌方历史副本
弱点：隐私模式、try-catch
```

战报：

```txt
Recall 快照眼记住了你的上一手操作。
这次它用在了你身上。
```

### 开始菜单广告商

```txt
定位：占格怪
速度：慢
血量：中
技能：在格子上放广告贴纸，占用玩家种植位
弱点：adblock、清理缓存
```

战报：

```txt
开始菜单广告商为你推荐了一个你没想装的应用。
```

### Windows 8 磁贴墙

```txt
定位：盾牌怪
速度：慢
血量：高
技能：挡住身后敌人，被打碎后掉出小磁贴
弱点：classic shell、sudo 锤
```

战报：

```txt
Windows 8 磁贴墙铺满了整条路线。
玩家短暂失去了开始按钮。
```

## 克制关系

| 怪物 | 主要威胁 | 推荐克制 |
| --- | --- | --- |
| Windows 更新兵 | 停顿一行植物 | print、for-print |
| 强制重启车 | 快速突破 | if 墙、rollback |
| Edge 传教士 | 转化植物 | sudo、try-catch |
| Teams 会议怪 | 范围停火 | try-catch、函数炮 |
| OneDrive 同步怪 | 分裂复制 | for-print、regex |
| 蓝屏巨人 | boss 短路 | scan、try-catch、函数炮 |
| 99% 卡住怪 | 高血量拖时间 | rollback、函数炮 |
| GWX 弹窗 | 弹窗干扰 | try-catch、esc |
| 1809 文件吞噬者 | 降级植物 | git backup |
| Clippy | 攻击错目标 | esc、try-catch |
| Recall 快照眼 | 复制玩家套路 | 隐私模式 |
| 开始菜单广告商 | 占用格子 | adblock |

## 怪物波次组合

第一版波次建议：

```txt
第 1 波：Windows 更新兵
第 2 波：Windows 更新兵 + 强制重启车
第 3 波：Edge 传教士 + Windows 更新兵
第 4 波：Teams 会议怪 + OneDrive 同步怪
第 5 波：蓝屏巨人 + 少量强制重启车
```

这套组合能测试玩家是否会建经济、补防御、用 try-catch、防快怪和集中火力打 boss。
