# 07 梗库与资料来源

## 用法

这个文件用于积累素材。每条素材都可以转成怪物、植物、关卡事件或战报。

原则：

```txt
真实事件只作为灵感
游戏里做夸张化表达
不要把资料写得太严肃
优先选玩家一眼能懂的梗
```
